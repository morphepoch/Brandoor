# Brandoor

Brandoor 是一个于 2026 年 6 月 9 日开始开发的全新品牌类 Agent 项目。

当前仓库仅用于展示可交互 HTML 预览页。项目才开发 1 天，因此这里不包含完整工程源码、构建配置、原始素材、PDF 资产或开发过程文件。

## 产品方向

Brandoor 面向快消、品牌市场和策略团队，目标是把品牌资料、视觉资产、策略模型和模板市场连接成一个品牌 Agent 工作台。

核心能力包括：

- 上传 PPT、PDF、KV 图片和品牌资料
- 自动解析品牌视觉 DNA，包括色彩、字体、版式、装饰元素和调性标签
- 沉淀品牌资产库，按色卡、字体、版式、图形组件和策略图表分类复用
- 基于模板框架与风格引擎生成 HTML 全案、PPT 策划案、产品手册和品牌视觉资产
- 支持 Agent 执行流、生成预览和下载
